---
## What's sumre ?
一個可以在臉書上使用的聊天機器人

----
### Bot 的好處
1. 透過messenger聊天泡泡可快速開啟
2. 在聊天視窗可輕易選擇想要的功能

----
### 他可以做什麼？
bot只是一個工具，就像遠端一個朋友，有事情你問他他就可以回答你。
但超出他能力的東西東西他就沒辦法幫你。
我們給了 sumre 數種小功能，完成我們生活中零星的小事。
